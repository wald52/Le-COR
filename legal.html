2. Politique de confidentialité

Ce site est conçu dans le respect de votre vie privée :

  • Aucune collecte de données personnelles : le site ne collecte, ne stocke et ne traite aucune donnée personnelle vous concernant.
  • Pas de cookies de traçage : aucun cookie publicitaire ou analytique n'est utilisé.
  • Traitement local : tous les graphiques, calculs et simulations s'exécutent directement dans votre navigateur. Aucune donnée ne quitte votre appareil.
  • PWA (application hors-ligne) : si vous installez le site en tant qu'application, les fichiers sont mis en cache localement via un Service Worker pour permettre l'utilisation hors connexion. Ces données restent sur votre appareil.

3. Propriété intellectuelle

Les données utilisées proviennent des rapports officiels du Conseil d'orientation des retraites (COR), de l'INSEE, de la Dares et de la Banque de France — sources publiques librement reproductibles. Le code source du site est ouvert : dépôt GitHub. Les textes éditoriaux et la mise en forme graphique sont la propriété de l'éditeur ; toute reproduction est soumise à autorisation préalable.

4. Responsabilité

Ce site est un projet citoyen indépendant, à but strictement pédagogique. Il n'est affilié ni au COR ni à aucune institution publique ou privée. Les graphiques et analyses sont construits à partir de données officielles reproduites fidèlement ; ils visent à les rendre accessibles, non à se substituer aux publications du COR. L'éditeur ne saurait être tenu responsable de toute décision prise sur la base de ces informations.
